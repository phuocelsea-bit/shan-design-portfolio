# SHAN.DESIGN Portfolio

静态作品集页面，包含：

- 首页视觉展示
- 作品展示与视频展示
- 学习经历与工作经验
- 专业技能、关于我、联系我

## 本地预览

```bash
node server.js
```

默认本地地址：

- `http://127.0.0.1:4173/index.html#home`

局域网访问：

- `http://你的局域网IP:4173/index.html#home`

## 自动部署

项目包含 GitHub Pages 自动部署工作流：

- 推送到 GitHub 仓库默认分支后
- GitHub Actions 会自动发布到 Pages
